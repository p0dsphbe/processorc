{ 引言：世界，早安 }

　　每个人都有那么一个时刻，想听从心底*真实的声音，行走、做梦、找自己……有些人把这个念头埋藏在繁忙与琐事的背后，慢慢地就忘记了；有些人记住了，某一天就出发了。我是MR.morning。为了更好地看看这个世界，我辞掉了大公司的工作，用自己的脚步和画笔去做一次长期旅行。旅行对我来说，是一种生活方式，更是一种心态。陪伴我的是一个空白的本子，上面记录了旅途中不期而遇的灵感。克里希那穆提曾经说过，多少世纪以来，我们被我们的老师、尊长、书本和圣人用汤匙喂大。我们总是说：“请告诉我，那高原深山及大地的背后是什么？”我们总是活在别人口中的世界，活得肤浅又空虚。所以，我想亲自看看我所在的这个世界，而不是道听途说。我想亲自去感受和认识，用心去行走。以福建厦门为始，经深圳、珠海，到云南丽江、香格里拉、大理，再到湖北武当山，江西景德镇、婺源，而后又到广西桂林阳朔，一路上参加了很多活动，和许多有趣的人合作了很多东西。而这一路陪伴我的，除了画画的本子，还有很多记忆，以及一些新鲜的想法和灵感。幸运的是，这些画带给很多人快乐和共鸣。在这些人里，有人提议这些画可以用来做更多有益的事情，比如可以帮慈善组织筹款，等等。而直到这时，我才发现，认识、理解、尊重自己所在的土地以及它上面发生的一切，并且能与自己的生活和周遭环境达成和解，其实是很多人的愿望。只是，单单落在纸上还不够，创作不能仅仅只是局限于熟悉的领域与材质，也不能只是满足于惯性和自我表达，要尝试让自己更加多元化。如何让自己的创作更加立体化，并且通过自己的创作去影响更多的人，去帮助他们实现愿望？我想，是时候去发现另一个自己，开启另一段“心的行走”了……

　　序：心中有画，面前就有路 }

　　关于这次辞职旅行、边走边画，其实我想了很长时间，在决定之前还花了一年多时间来思考到底什么才是我想要的。我做过房地产广告，做过家纺，学过服装设计，甚至做过游戏美术，但是这些对我来说，都只能算是一种到达自己的过渡缓冲，我从不后悔做过这些事。凡事都有因有果，也许正是因为这些经历，才让我更加确定什么才是自己需要的。

　　我还是想画画，画自己的画，做自己喜欢的东西。我相信不管哪一条路，只要用心走、努力走，都不会惨到真的养不活自己的地步。

　　也许边旅行边画画对于很多人来说，是一个不能实现的梦想，而有一天当你真的这么做了之后才会发现，其实在国内外一直都有很多人选择这样的生活方式。钱、安全、家庭，诸多的担心，根本没有想象的那么严重。有手有脚的健康人，总会活下去，并且真的不会活得很差，只要你肯努力。旅途中靠自己双手攒路费的人多的是，每一分钱你都会挣得清清楚楚、开开心心。

　　毕竟，很多事情不会自己完全准备好，放到你面前等你去做的。有时间时没心情，有心情时又没时间，那么当你既有心情又有时间的时候，何不赶紧出发？

　　有很多人觉得旅行是逃避社会、逃离生活；相反，我却觉得这才是接触生活和社会*好的方式。每天要面对形形色色、来自四面八方的人，要处理自己吃喝拉撒睡等事宜，流水似的萍水相逢，随时要准备好背包去往下一站的重新开始，都不比日复一日在同一所公司、同一间公寓的工作、生活更简单。

　　自由才是一个人年轻时的*大资本，浪费了多可惜。大千世界的种种颜色还都没有见过，却要将自己彻底染上职场的高级灰，无论如何我都觉得划不来。人只能不可逆地活一次，而我想活得更划算些。

　　可计量的房子、车子都是可以用钱来解决的，而很多不可计量的东西，比如见识、智慧、友情、梦想、健康等，都是用钱解决不了的。钱没了可以再挣，但其他东西没了就没了。

　　你说是不是？
